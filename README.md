# Сайт-навигатор для кафедры СГН3 МГТУ им. Баумана

1) Ссылка на опубликованный сайт - https://sgn3-info.tilda.ws/bmstu_sgn3

2) Проект реализован на платформе Tilda Publishing.
Редактирование: Для внесения изменений в структуру, тексты, изображения или блоки необходимо зайти в личный кабинет на https://tilda.ru В редакторе Tilda доступно визуальное редактирование элементов.
Доступ: Поскольку сайт создан в облачном конструкторе, локальная версия отсутствует. Можно просмотреть сайт по ссылке (п1).
